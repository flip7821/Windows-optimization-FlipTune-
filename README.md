**FLIPTUNE** – мощная консольная утилита для тонкой настройки Windows.  
Содержит **70+ проверенных твиков** (приватность, производительность, сеть, игры, очистка и даже ультра-низкая задержка).  

✅ **Откат каждого изменения** (реестр + службы)  
✅ **Профили/пресеты** – сохраняй и применяй наборы твиков  
✅ **Эксперт-режим** – доступ к расширенным и опасным опциям  
✅ **Dry-run** – посмотри, что будет сделано, без реальных изменений  
✅ **Работает без установки** – один .exe файл

## 🚀 Быстрый старт

1. **Скачай** `FLIPTUNE.exe` из раздела [Releases](https://github.com/flip7821/Windows-optimization-FlipTune-/releases/tag/v1).
2. **Запусти от имени администратора** (правой кнопкой → «Запуск от имени администратора»).
3. В главном меню выбери нужный пункт (например, `1` – Performance Mode, `4` – Gaming Mode, `9` – только игровые твики).
4. Для безопасного эксперимента сначала включи **Dry‑run** (клавиша `D`) – программа покажет, что будет сделано, но ничего не изменит.
5. Перед первым применением создай **точку восстановления** (пункт 14) и **бэкап реестра** (15).

## 🔄 Откат изменений

**Каждый твик автоматически сохраняет снепшот** реестра и состояния служб перед выполнением.

- `16` – отменить **последний** применённый твик  
- `17` – показать историю отката (стек)  
- `19` – сбросить **все** твики из папки `FLIPTUNE_Backup` (требуется подтверждение)

## 💾 Профили (пресеты)

Сохрани свой идеальный набор твиков, чтобы применить его в один клик:

1. Примени нужные твики вручную.
2. Зайди в меню `18` → выбери `1` (Save current session as profile) → введи имя профиля.
3. В следующий раз: `18` → `2` (Load & apply a saved profile) → выбери профиль.

Профили хранятся в папке `FLIPTUNE_Profiles` рядом с `FLIPTUNE.exe` (формат JSON).

## ⚠️ Ложные срабатывания антивирусов

**Почему антивирусы ругаются?**  
FLIPTUNE скомпилирован в `.exe` с помощью утилиты `PS2EXE`. Эта обёртка иногда используется злоумышленниками для упаковки вредоносных скриптов, поэтому многие антивирусы (особенно Microsoft Defender, K7, SecureAge) дают **ложное срабатывание** на любой файл, созданный таким способом.

**Что делать?**  
- 🛡️ **Добавь в исключения** антивируса, если доверяешь автору.  
- 📄 **Исходный код полностью открыт** – ты можешь скомпилировать его сам, используя `PS2EXE` или просто запустить `FLIPTUNE.ps1` напрямую (без обёртки).

## 🛠 Сборка из исходников (для продвинутых)

Если не доверяешь готовому `.exe` или хочешь внести изменения:

1. Установи модуль `PS2EXE`:
   powershell
   Install-Module -Name PS2EXE -Force
2. Скомпилируй скрипт:

powershell
ps2exe -inputFile "path to file" -outputFile "C:\Users\your\Desktop\FlipTune Free\FLIPTUNE.exe" -iconFile "C:\Users\user\Desktop\FlipTune Free\fliptune.ico"

Также можно запускать напрямую .ps1 в PowerShell с правами администратора.
